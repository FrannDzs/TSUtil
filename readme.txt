
Hinweise zur Benutzung der TSUtil-Source.

Vorraussetzungen:
- Java-SDK ab Version 1.5 muss installiert sein (z.B. Windows: 'jdk-1_5_0_22-windows-i586-p.exe' )
	aber auch ALLE neueren Java-Versionen sollten funktionieren.

Installation:
- beliebiges Verzeichnis als Installationsverzeichnis anlegen und Zip-Datei dahin entpacken
- Zugriffspfad in Dos.lnk (Ausführen in) auf das Installationsverzeichnis aendern


Nutzung:

Zur Kompilierung Dos-Fenster oeffen (Dos.lnk) und Kommando 'tsutil' eingeben. Falls keine fehler gemeldet werden,
kann das Ergebnis direkt genutzt werden Dazu im Dos-Fenster das gewuenschte Kommando eingaben. z.B:
'java -Xmx256m TSUtil -l ichk <pfad>'
Die Log-Datei wird gemaess Beispiel im Installationsverzeichnis als 'TSutil.log' erzeugt.

Achtung: Bitte auf Gross-/Kleinschreibung achten
